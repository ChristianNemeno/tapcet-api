# Core Features (Minimum Viable Product)

## User Management
- Register, login, logout.
- Track user scores and quiz attempts.

## Quiz Management
- Admin (or you) can create quizzes.
- Each quiz has multiple questions.
- Each question has multiple choices.
- One choice is correct.

## Quiz Attempt
- User can start a quiz.
- Submit answers.
- Backend checks correctness and returns score.

## Results
- Store user’s past attempts and scores.
